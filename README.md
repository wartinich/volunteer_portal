# Volunter portal
Web portal for volunteers from all over Ukraine, which will help optimize the work of many people and organizations in difficult times.

## **Technologies stack:** 
Backend:
* Django
* Django REST framework
* Docker
* PostgreSQL

Frontend:
* React
* tailwind css
* react-router-dom
* axios


## **Setup Instructions**

### Clone repository

```bash
git clone <repo>
```

### Start Server

1) Create an image
```bash 
docker-compose build
```
2) Run container
```bash
docker-compose up
```
3) Go to url address
```link
http://127.0.0.1:8000/
```
